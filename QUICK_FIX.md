# Быстрое исправление ошибки 500

## Проблема: "Server Error" при оформлении заказа

### Причина
Переменные окружения для Telegram бота не установлены в файле `.env`.

### Решение (3 шага)

#### Шаг 1: Откройте файл `.env`

Найдите файл `.env` в корне проекта и откройте его в текстовом редакторе.

#### Шаг 2: Добавьте переменные

Добавьте в конец файла `.env` следующие строки:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_BOT_USERNAME=ваш_username_бота_без_@
```

**Где взять эти данные:**
1. Откройте Telegram
2. Найдите бота `@BotFather`
3. Отправьте `/mybots`
4. Выберите вашего бота
5. Скопируйте токен и username

**Пример:**
```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz-1234567890
TELEGRAM_BOT_USERNAME=restaurant_verification_bot
```

**ВАЖНО:** 
- Username должен быть БЕЗ символа `@`
- Не добавляйте пробелы вокруг знака `=`
- Не используйте кавычки вокруг значений

#### Шаг 3: Очистите кэш

Выполните в терминале:

```bash
php artisan config:clear
php artisan cache:clear
```

### Проверка

После выполнения шагов:

1. Обновите страницу в браузере (Ctrl+F5)
2. Попробуйте создать заказ снова
3. Ошибка должна исчезнуть

### Если ошибка осталась

1. Проверьте, что файл `.env` сохранен
2. Убедитесь, что переменные написаны правильно (без опечаток)
3. Выполните снова: `php artisan config:clear`
4. Проверьте логи: `storage/logs/laravel.log`

### Дополнительная помощь
